Bahasa Pemograman HTML
HTML adalah singkatan dari HyperText Markup Language, yaitu bahasa standar yang digunakan untuk membuat dan menyusun halaman web di internet. HTML menggunakan elemen-elemen atau tag-tag tertentu untuk mengatur struktur dan konten halaman web, seperti teks, gambar, tautan, tabel, dan lainnya. Dengan HTML, pengembang web dapat menentukan bagaimana informasi ditampilkan di browser web.
